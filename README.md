## Run Locally

Clone the project

```bash
  git clone https://github.com/sumansauravmay/Lens_Corp_Home_Page.git
```

Go to the project directory

```bash
  cd lens_home_page

```

Install dependencies for lens_home_page

```bash
  cd lens_home_page
  npm install
```
